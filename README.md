## 🖥️ Projeto

Front end de projeto products, consumindo api criada.

## 🧱 Tecnologias

+ Reactjs
+ Typescript
+ Styled Components
+ Axios


## 🏠 Executar localmente

Clone o projeto

```bash
git clone https://github.com/FabioPenedo/productsfrontend
```

Entre na pasta do projeto

```bash
cd products-front
```


Instale as dependências

```bash
npm install
```

Inicie o servidor

```bash
npm run dev
```
